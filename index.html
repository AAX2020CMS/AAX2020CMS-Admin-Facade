AAX 2020 CMS Admin Facade.

Semi automated admin GUI mockup generator

This is a semi-automated build system for trying out GUI templates and themes for the admin.

It works with Node.Js , npm , Grunt, Bower, Dart SDK/Pub and a few dependencies.

It reads from the /source directory and outputs to the /build directory, which should also be exposed in the gh-pages branch as /build/

It currently supports AngularUI + BootStrap / MaterialDesign ; may later include Zurb.
